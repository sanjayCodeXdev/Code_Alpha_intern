# Task 2: Chatbot for FAQs

> **CodeAlpha AI Internship · Task 2: Chatbot for FAQs**

A smart FAQ Chatbot that uses NLP (cosine similarity) to match user questions with the closest FAQ answer — with a clean chat UI.

---

## 🚧 Status: In Progress

This task is currently under development.

---

## 📋 Requirements (from CodeAlpha)

- Collect a dataset of FAQs on a specific topic/product
- Preprocess text (tokenize, clean) using NLTK or SpaCy
- Use cosine similarity or intent matching to find the best answer
- Display the matched answer to the user
- **Optional:** Build a simple chat UI

---

## 🛠️ Tech Stack (Planned)

| Technology | Purpose |
|---|---|
| Python | Backend / NLP logic |
| NLTK / SpaCy | Text preprocessing & tokenization |
| Scikit-learn | TF-IDF vectorizer & cosine similarity |
| Flask / Streamlit | Web UI for the chatbot |
